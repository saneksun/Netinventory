# NetInventory

NetInventory is a tool that is scanning network devices in order to collect hardware inventory information using standard SNMP (v1, v2c) and NETCONF (SSH port 830) protocols.

Supported platforms:
- Juniper JunOS
- Cisco IOS-XR

Prerequisites
- Django
- PostgreSQL
